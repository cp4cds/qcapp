This a data availability tool for Global Climate Projections provided to the Copernicus Climate Data store. The Copernicus Climate Change Service (C3S) Climate Data Store make available a quality controlled subset of CMIP5 data. This tool allows users to see what data is available.

A user can interrogate the data by variable or model. A more complex query can submitted using the data availability matrix. The data availability matrix filter allows users to search the CMIP5 subset for one or more variables within one or more experiments simulataneously.
